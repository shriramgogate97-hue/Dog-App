# Dog Rescue Network – Product Plan

## Vision
Create a structured rescue ecosystem so animal incidents are reported quickly, assigned to trusted NGOs, and tracked transparently.

## Core Roles

1. **Customer/Citizen**
   - Raise SOS incident with image/video and GPS.
   - Track rescue status.
   - Earn points for verified helpful reports.

2. **NGO Team**
   - Receive nearby incidents.
   - Accept/decline dispatch.
   - Update rescue ETA and outcomes.

3. **Admin**
   - Manage NGO onboarding.
   - Moderate incident quality.
   - Track heatmaps, incident categories, and SLA.

## MVP Features

- Emergency SOS button.
- Incident upload (photo/video + description + location).
- Rescue team assignment and live status tracking.
- Reward points for users.
- Animal profile cards (vaccination, sterilization, treatment history).
- Mobile + web dashboard with sidebar navigation.

## Suggested Tech

- **Frontend web:** ReactJS (Vite).
- **Mobile apps:** Flutter.
- **Backend (next phase):** Node.js + Express + PostgreSQL + Firebase Cloud Messaging.
- **Maps:** Google Maps Platform or OpenStreetMap stack.

## Future Analytics

- Incident hotspot map by ward/city.
- Most common case type by season.
- Average rescue arrival time by NGO.
- Vaccination and sterilization campaign coverage.
